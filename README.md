# 🌍 Climate Analysis — Étude du réchauffement climatique

## 🎯 Objectif du projet

Analyser l’évolution des températures terrestres depuis 1880 afin de mettre en évidence les tendances du réchauffement climatique et les disparités géographiques.

---

## 📦 Contenu du projet

Ce projet est composé de 3 parties :

### 📊 1. Analyse des données (Python / Jupyter Notebook)

* Nettoyage et exploration des données
* Création de visualisations
* Analyse des tendances temporelles et géographiques
  
📁 [Code_Python_Température_Terrestre.ipynb](https://github.com/DaliaB19/climate-analysis/blob/main/Code_Python_Température_Terrestre.ipynb)

---

### 📄 2. Rapport complet (PDF)

Un document de synthèse présentant :

* les analyses principales
* les graphiques clés
* les conclusions du projet
  
📁 [Projet Température Terrestre_Analyse.pdf](https://github.com/DaliaB19/climate-analysis/blob/main/ProjetTemp%C3%A9ratureTerrestre_%20Analyse.pdf)

---

### 📊 3. Application interactive (Streamlit)

Une application permettant de visualiser les résultats de manière interactive pour une présentation orale.

📁 [streamlit_fin.py](https://github.com/DaliaB19/climate-analysis/blob/main/streamlit_fin.py)

---

## 📈 Résultats clés

* Confirmation d’une tendance globale au réchauffement climatique
* Différences marquées entre hémisphère Nord et Sud
* Mise en évidence de variations régionales importantes

---
